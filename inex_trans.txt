MOVJ():
Команда, предназначенная для перемещения робота по прямой от точки к точке вне рабочей траектории. Движение происходит с максимальной скоростью, прямо пропорциональной значению параметра VJ. Данная инструкция используется в случаях, когда необходимо направить робота в координаты точки начала работы.
Для данной инструкции указываются следующие параметры:
    P(GP) - целевая локальная(глобальная) точка 
    VJ - скорость движения. Указывается в диапазоне 1%...100%.
    PL - Коэффициент сглаживания. 
    ACC - Коэффициент ускорения. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, равное V. 
    DEC - Коэффициент замедления. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, равное V. 
    TIME - Значение временного интервала в милисекундах, оставшееся до конца выполнения текущей инструкции, указывающее начало предварительного выполнения следующей инструкции(команды).
Например:
    MOVJ P001 VJ = 10 PL = 0 ACC = 10 DEC = 10 0

MOVL():
Команда прямолинейного движения робота в зоне проведения работы.
Для данной инструкции указываются следующие параметры:
    P(GP) - целевая локальная(глобальная) точка 
    V - скорость движения. Указывается в мм/сек. Выбирается диапазон значений 1...1000.
    PL - Коэффициент сглаживания. 
    ACC - Коэффициент ускорения. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    DEC - Коэффициент замедления. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    TIME - Значение временного интервала в милисекундах, оставшееся до конца выполнения текущей инструкции, указывающее начало предварительного выполнения следующей инструкции(команды)
Например:
    MOVL P001 V = 200 mm/s PL = 0 ACC = 20 DEC = 20 0

MOVC():
Перемещение робота по дуге 180 градусов, обозначенной тремя точками.
Следует помнить, что корректный алгоритм отработки указанного движения включает в себя как минимум три команды - MOVJ()/MOVL(), MOVC(), MOVC() и выглядит следующим образом:
Для перемещения к первой точке дуги необходимо использовать команду MOVJ() или MOVL().
Перемещение ко второй точке дуги происходит посредством команды MOVC(). 
Для данной инструкции указываются следующие параметры:
    P(GP) - локальная(глобальная) точка 
    V - скорость движения. Указывается в мм/сек. Выбирается диапазон значений 1...1000.
    PL - Коэффициент сглаживания. 
    ACC - Коэффициент ускорения. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    DEC - Коэффициент замедления. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    TIME - Значение временного интервала в милисекундах, оставшееся до конца выполнения текущей инструкции, указывающее начало предварительного выполнения следующей инструкции(команды).
Перемещение к третьей точке дуги происходит также посредством команды MOVC(). При этом, параметры V, PL, ACC, DEC, TIME должны соответствовать параметрам предыдущей инструкции MOVC(). 
Например:
    MOVJ P001 VJ = 10% PL = 0 ACC = 1 DEC = 1 0
    MOVC P002 V = 100 mm/s PL = 0 ACC = 10 DEC = 10 0
    MOVC P003 V = 100 mm/s PL = 0 ACC = 10 DEC = 10 0

MOVCA():
Робот проходит полную окружность, обозначенную тремя точками.
Следует помнить, что корректный алгоритм отработки указанного движения включает в себя как минимум три команды - MOVJ()/MOVL(), MOVCA(), MOVCA().
Если сварка(обработка) происходит инструментом, отличном от штатного, закрепленным в манипуляторе робота, перед переходом к первой точке окружности необходимо выбрать заранее заданную в настройках (Settings -> Tool Calibration) систему координат этого инструмента:
Меню программы -> Insert -> Coordinate Switch Class -> SWITCHTOOL -> [number of tool].
Далее необходимо произвести переход к первой точке окружности посредством команды MOVJ() или MOVL(). 
После того, как произведен переход к первой точке окружности, необходимо указать вторую точку окружности. Это производится при помощи инструкции MOVCA().
Для данной инструкции указываются следующие параметры:
    P(GP) - локальная(глобальная) точка 
    V - скорость движения. Указывается в мм/сек. Выбирается диапазон значений 1...1000.
    PL - Коэффициент сглаживания. 
    ACC - Коэффициент ускорения. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    DEC - Коэффициент замедления. Выбирается из диапазона 1%...100%. Рекомендовано указывать значение, вычисляемое по формуле V*0,1 
    TIME - Значение временного интервала в милисекундах, оставшееся до конца выполнения текущей инструкции, указывающее начало предварительного выполнения следующей инструкции(команды) 
    SPIN - Режим вращения:
        Attitude unchanged (0) - прохождение траектории в соответствии с настройками режима калибровки Р0001(инструмент, установленный на робот по умолчанию).  
        6-axis non rotating (1) - При прохождении траектории горелка остается в одном и том же положении, оказываясь в зависимости от текущего положения то внутри, то снаружи провариваемого периметра.
        6-axis rotating (2) - При прохождении траектории горелка (6 ось) вращается таким образом, что ее положение (внутри или снаружи периметра проварки) остается неизменным. Достигается путем дополнительных изменений координат по остальным осям. При использовании этого режима стоит помнить, что изменение значений по 6 оси находится в диапазоне -200...+200 градусов. Поэтому выставлять точки окружности для такого режима необходимо так, чтобы значения координаты 6 оси изменялись от +180 до -180 градусов или наоборот, от -180 до +180. Например, в первой точке значение координаты 6 оси составляет -180, во второй - 0, а в третьей - +180 градусов. 
Третья точка окружности также указывается инструкцией MOVCA(). При этом, параметры V, PL, ACC, DEC, TIME, SPIN должны соответствовать параметрам предыдущей инструкции MOVCA(). 
Например:
    MOVJ P001 VJ = 10% PL = 0 ACC = 1 DEC = 1 0
    MOVCA P002 V = 100 mm/s PL = 0 ACC = 10 DEC = 10 SPIN = 1 0
    MOVCA P003 V = 100 mm/s PL = 0 ACC = 10 DEC = 10 SPIN = 1 0
